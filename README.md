## Incremental-Learning


## Project Structure

The project is organized into the following main directories:

```
Thesis/
├── 01_Protocol/                     # Protocolo de la tesis
│   ├── build/                       # Archivos generados durante el proceso de compilación
│   ├── infoToStart/                 # Información adicional para comenzar
│   └── main.tex                     # Documento principal en LaTeX
├── 02_References/                   # Artículos, citas y bibliografía
│   └── dataset.bib                  # Archivo de bibliografía en formato BibTeX
├── 03_Chapters/                     # Capítulos de la tesis
│   └── approach.tex                 # Enfoque de la tesis en LaTeX
├── 04_Papers/                       # Artículos utilizados para la tesis
│   ├── General_Papers/              # Artículos generales
│   └── Protocol_Papers/             # Artículos relacionados con el protocolo
├── 05_Images/                       # Imágenes utilizadas en la tesis
├── 06_Code/                         # Código fuente del proyecto
│   ├── C_Code/                      # Código en C
│   ├── Notebooks/                   # Notebooks Jupyter
├── 07_Thesis/                       # Versiones de la tesis
│   ├── Drafts/                      # Borradores de la tesis
│   ├── Final/                       # Versión final de la tesis
│   ├── OldVersions/                 # Versiones anteriores
│   ├── Revisions/                   # Revisiones de la tesis
├── 08_Presentations/                # Presentaciones realizadas
│   └── 14vo_Colloquium_UAEMex_2022/ # Presentaciones del Coloquio 2022
├── 09_Resources/                    # Recursos adicionales
│   ├── Algorithms/                  # Recursos sobre algoritmos
│   ├── IncrementalLearning/         # Recursos sobre aprendizaje incremental
│   ├── LaTeX_Guides/                # Guías de uso de LaTeX
├── README.md                        # Archivo README principal
└── .git/                            # Carpeta oculta de Git


```
